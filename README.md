## About The Project

Nothing Compass is designed to be a simple, distraction-free tool for navigation and orientation. The user interface is specifically crafted to harmonize with the **Nothing Phone UI aesthetics**, featuring a minimalistic design that respects your device's theme settings.

### Features

*   **Nothing Aesthetics**: UI specifically designed to match the Nothing OS look and feel.
*   **Compass**: Precise directional information with bearing.
*   **Inclinometer**: Measure slope, tilt, and angles with ease.
*   **Theme Integration**: Automatically adapts to your system's light or dark mode preferences.
*   **Haptic Feedback**: Optional tactile feedback for a more responsive experience.
*   **Zero Ads**: A completely clean experience without interruptions.

### Built With

*   [Android SDK](https://developer.android.com/studio)
*   [Java](https://www.oracle.com/java/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Android Studio Arctic Fox or later
*   JDK 11 or higher
*   Android SDK 21+

### Installation

1.  Clone the repository
    ```sh
    git clone https://github.com/iso53/Nothing-Compass.git
    ```
2.  Open **Android Studio**.
3.  Select **Open an existing Android Studio project**.
4.  Navigate to the cloned directory and select it.
5.  Wait for Gradle debugging to complete.
6.  Run the app on your emulator or physical device.
